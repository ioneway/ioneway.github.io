隐藏NavigationBar下边线

有些app需要将NavigationBar的下边线隐藏，从而使得页面看起导航条和下面的页面融为一体。

寻找下边线View视图

- (UIImageView*)bottomLineView:(UIView*)view {

    if([view isKindOfClass:UIImageView.class] && view.bounds.size.height<=1.0) {
        return(UIImageView*)view;
    }
    for(UIView*subview in view.subviews) {
        UIImageView*imageView = [self bottomLineView:subview];
        if(imageView) {
            return imageView;
        }
    }
    return nil;
}

将该函数作为UINavigationBar的扩展函数即可直接调用得到盖视图，然后在希望隐藏的地方将hide属性值为YES

应用场景：

1，当页面视图是一个UIScrollView时，在正常状态下隐藏下边线，在用户向上滑动时候显示下边线。
